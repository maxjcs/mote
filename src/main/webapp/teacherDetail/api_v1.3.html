教师主页联调接口说明（教师端v1.3）

Created by Feil.Wang on 2015/7/9.

一、教师主页URL

  1. 【NEW】教师主页URL传参规则：

    URL：http://{{url}}/teacherDetail/teacherCode.html?teacherId={{teacherId}}&version=1.3

    说明：version表示H5页面版本号，version等于1.3时，表示教师端v1.3浏览界面，不传version参数则表示第三方（WEB分享）界面。

    187环境测试URL: http://192.168.1.187:8080/teacher/teacherDetail/teacherCode.html?teacherId=28dfe9a6-1133-4a6e-850b-77b89a62277d&version=1.3

二、HTML提供给APP的接口

  1. 【NEW】刷新留言：jsInvoke ("refreshMessage")

    说明：删除留言后或者返回到教师个人主页时可调用此方法刷新主页的留言列表

三、APP提供给HTML的接口

  1. 点击二维码：window.external.goToQRCodeView()

    说明：页面点击二维码图标跳转到二维码界面的方法

  2. 跳转至习题详情：window.external.workbookDetail(id)

    说明：点击习题集标题跳转至习题详情，参数为习题集id

  3. 【NEW】跳转至习题集列表页：window.external.workbookList()

    说明：点击习题集模块中的“查看全部习题集>>”跳转至习题集列表页

  4. 【NEW】跳转至留言列表页：window.external.goMessageList()

    说明：点击页面底部“查看全部留言>>”跳转至留言列表页

  5. 【NEW】删除留言：window.external.deleteMessage(id)

    说明：参数id为该条留言的id

  6. 【NEW】跳转到指定留言的详情页：window.external.goMessageDetail(id)

    说明：点击单条留言，跳转到这条留言的详情页，参数id为该条留言的id
