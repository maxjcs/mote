教师主页联调接口说明（教师端）

一、教师主页URL

1、教师主页URL传参规则：http://{{url}}/teacherDetail/teacherCode.html?teacherId={{teacherId}}&viewMode={{viewModeValue}}

说明：viewMode表示那种浏览模式，viewMode等于1时，表示教师端浏览界面，不传viewMode参数则表示第三方（WEB分享）界面。

187环境测试URL: http://192.168.1.187:8280/teacher/teacherDetail/teacherCode.html?teacherId=157dcab6-9395-4432-9596-d7d8f4d9c6f7&viewMode=1

二、统一调用方法：jsInvoke

用法：jsInvoke("事件类型",数据)

1. 发送留言：sendLeaveMessage

说明：留言输入框部分由native实现，APP发送留言成功后，调用此方法，刷新留言部分的DOM

jsInvoke ("sendLeaveMessage",{ content:"test message!!", //留言内容 createTime: 1432894115000 //发送时间 })

三、HTML需要APP提供的接口

1.点击二维码：window.external.goToQRCodeView()

说明：页面点击二维码图标跳转到二维码界面的方法

2.点击习题集标题跳转至习题详情：window.external.workbookDetail(id)

说明：参数为习题集id
